To run the Scholarship Application:
1) Make sure you have python installed.
2) In the command prompt, run the command <br>
pip install django 
3) To verify the installations, run the command <br>
django-admin --version <br>
in the command prompt.
4) Open the code and in manage.py file terminal, run command <br>
python manage.py runserver
5) Paste the url in web browser and its done.
